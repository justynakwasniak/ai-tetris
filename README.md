# ai-tetris
A React-based Tetris game that uses AI to introduce novel features.
**********

# My Project

## Setup
1. Clone the repo
git clone <repo-url>

2. Install dependencies
npm install

3. Start dev server
npm run dev

## Linting & Formatting
- Run ESLint:
npm run lint

- Run Prettier:
npm run format


## Folder structure
- `src/components` - React components  
- `src/hooks` - Custom hooks  
- `src/utils` - Utility functions
